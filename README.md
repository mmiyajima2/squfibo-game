# SquFibo（すくふぃぼ）

**置いた一手で「気づけた子」が得をする、暗算とひらめきの数字ゲーム**

## 概要

SquFiboは、小学生向けの2人対戦ブラウザゲームです。3×3の盤面に数字カードを配置し、特定の組み合わせ（役）を作って星を獲得します。暗算力よりも「盤面全体を見渡して気づく力」を重視した、教育的かつ戦略的なゲームです。

## ゲームの特徴

- **気づきを重視**: 役の自動判定は行わず、気づいたプレイヤーだけが申告できる
- **脱落なし**: 手札が0枚になっても脱落せず、ゲームを続けられる
- **シンプルなルール**: 3×3の盤面と4種類の数字カードのみ
- **戦略性**: 盤面をどう相手に渡すか、読み合いが生まれる
- **デジタル/物理対応**: ブラウザゲームとしても、物理ボードゲームとしても遊べる設計

## ゲームルール概要

### 使用要素
- **数字カード**: 1, 4, 9, 16（各赤8枚・青8枚、合計64枚）
- **盤面**: 3×3のマス
- **星**: 34個（得点トークン）

### ゲームの流れ
1. 各プレイヤーに手札13枚を配る
2. 交互に手札から1枚を盤面に配置
3. 役が成立したら申告し、星を獲得
4. 星がすべて獲得されるか、山札が0枚になったらゲーム終了
5. 星が多いプレイヤーが勝利

### 役の種類（同色のみ）
- **1 + 4** または **4 + 9**: 2枚除去、2枚ドロー、星2個
- **1 + 4 + 16**: 3枚除去、3枚ドロー、星3個（直線・L字型のいずれも可）

詳細なルールは [docs/spec_ja.md](docs/spec_ja.md) を参照してください。

## セットアップ

### 必要環境
- Node.js 18以上

### インストール
```bash
npm install
```

### 開発サーバーの起動
```bash
npm run dev
```

### ビルド
```bash
npm run build
```

### プレビュー
```bash
npm run preview
```
